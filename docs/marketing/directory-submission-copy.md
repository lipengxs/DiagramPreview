### Full Description

```text
DiagramPreview is a browser-based toolbox for previewing, debugging, converting, and exporting visual scripts and structured text.

It supports Mermaid, PlantUML, Graphviz, D2, draw.io XML, Markdown with Mermaid, JSON, YAML, XML, CSV, SQL schemas, OpenAPI, GraphQL, Protobuf, DBML, Prisma, Docker Compose, Kubernetes manifests, Terraform, GitHub Actions, Helm values, Nginx configs, Grafana dashboards, Prometheus alert rules, logs, traces, package.json files, regex patterns, mind maps, and sequence diagrams.

The tool is especially useful when working with AI-generated diagrams or configuration snippets. Large language models can generate Mermaid, PlantUML, draw.io XML, dashboards, alerts, and architecture descriptions, but they usually do not provide a reliable visual preview. DiagramPreview closes that gap by letting users paste source text, preview the result, fix errors, export images, or download editable files.

It is designed for developers, DevOps engineers, product managers, architects, documentation teams, and anyone who needs to turn text-based technical content into something visual, inspectable, and easier to understand.
```

这个工具特别适合 AI 生成图表和技术文档的场景。现在很多大模型可以生成 Mermaid、PlantUML、draw.io XML、监控看板、告警规则和架构描述，但通常缺少可靠的在线预览和调试能力。DiagramPreview 解决的就是这个问题：用户可以粘贴源码，立即看到预览效果，检查错误，导出图片，或者下载可编辑文件。

它适合开发者、DevOps 工程师、产品经理、架构师、技术写作者和文档团队使用，帮助他们把文本形式的技术内容变成更直观、可检查、可分享的视觉结果。
