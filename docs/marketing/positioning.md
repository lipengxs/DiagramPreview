# Positioning

DiagramPreview is a browser-first diagram preview, conversion, and export toolbox for developers, technical writers, and AI-assisted documentation workflows.

## Core Angle

AI tools can already generate Mermaid, PlantUML, architecture notes, JSON, YAML, OpenAPI, SQL, Docker Compose, Kubernetes manifests, and other structured text.

The missing step is usually visual validation:

- Large language models can generate diagram code, but they rarely provide a stable preview.
- Generated Mermaid or PlantUML often has small syntax problems.
- Developers still need SVG/PNG/PDF export before putting a diagram into README, PRD, architecture docs, proposals, or blog posts.
- Some generated outputs need to become editable files, such as draw.io/diagrams.net XML.

DiagramPreview focuses on that gap: paste generated or handwritten diagram text, preview it, fix it, and export it.

## Target Audience

- Developers writing README, design docs, API docs, architecture proposals, and incident reviews.
- Technical writers and product managers who need quick diagrams without a heavyweight desktop tool.
- AI-assisted builders who ask Gemini, DeepSeek, ChatGPT, Claude, or Cursor to generate diagrams.
- DevOps and platform engineers working with Kubernetes, Docker Compose, Grafana, Prometheus, OpenAPI, SQL, and dependency maps.
